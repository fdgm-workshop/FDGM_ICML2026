# Foundations of Deep Generative Models

Static website for the ICML 2026 workshop:

**Foundations of Deep Generative Models: Understanding Memorization, Generalization, and Reasoning**

## Run locally

This site is plain HTML and CSS. There is no build step.

```powershell
cd FDGM_ICML2026
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Main files

- `index.html` - page structure and workshop content
- `styles.css` - visual design and responsive layout
- `seoul.jpg` - hero banner image
- `figures/` - local organizer and speaker headshots used by the page

## Notes

- The workshop schedule on the site reflects the proposed full-day format from the workshop materials.
- Workshop contribution deadlines and submission instructions are intentionally left for later announcement instead of carrying over stale ICLR template dates.
- Forked from DeLTa 2026 and unused template assets have been removed so the repository only contains website files and images referenced by the site.
